## 🚀 Tecnologias

Esse repositório contém essas tais tecnologias :

- HTML
- CSS
- TypeScript
- Node.Js
- Express
- Socket.io

<br>

  ## 💻 Projeto

  Esse projeto foi desenvolvido para eu poder aprender as utilidades de fazer um Chat em tempo Real, utilizando conceitos
  de Websockets e comunicação bidirecional.

  <br>


  ## ⚙️ Instalações
  <br>

  🔗 **Clone o repositório :**

  ```bash
    https://github.com/betafontes/real-time-chat.git
  ```

  🔗 **Entre no diretório :**

  ```bash
    cd real-time-chat
  ```

 🔗 **Instale as dependências :**

 ```bash
   npm install
 ```

🔗 **Inicie o servidor :**

 ```bash
   npm run dev
 ```

   
🖼️ **Deploy :**

```bash
  http://localhost:3000.
```
